Azure ACR Dynamo Container Registry Access

Registrydynamoci.azurecr.io/ai-dynamo/dynamo
Architecturesamd64 only
Auth methodaz login (ACR authentication guide, DL: access-azure-dynamo-engineer)
Also available viaAWS ECR (includes arm64)

Prerequisites (one-time)

Install Azure CLI and get access. See ACR authentication guide (DL: access-azure-dynamo-engineer).

# Install Azure CLI
curl -sL https://aka.ms/InstallAzureCLIDeb | sudo bash

Step 1: Login

# Login to Azure (opens browser)
az login

# Login to ACR (token valid for 8 hours)
az acr login --name dynamoci

Step 2: List available images

# List latest tags
az acr repository show-tags --name dynamoci \
  --repository ai-dynamo/dynamo \
  --orderby time_desc --top 20 --output table

Step 3: Pull an image

ACR tags follow the pattern: <commit_sha>-<framework>-<type>-cuda<ver> (no arch suffix — all amd64).

# Pull a vLLM dev image (CUDA 12)
docker pull dynamoci.azurecr.io/ai-dynamo/dynamo:<commit_sha>-vllm-dev-cuda12

# Pull a TRT-LLM runtime image (CUDA 13)
docker pull dynamoci.azurecr.io/ai-dynamo/dynamo:<commit_sha>-trtllm-runtime-cuda13

# Pull an SGLang dev image (CUDA 13)
docker pull dynamoci.azurecr.io/ai-dynamo/dynamo:<commit_sha>-sglang-dev-cuda13

Step 4: Pull + build local-dev

(ACR=dynamoci.azurecr.io/ai-dynamo/dynamo:<tag> \
 && docker pull $ACR \
 && curl -sL http://speedoflight.nvidia.com/dynamo/dynamo-utils/container/build_localdev_from_dev.py \
    | python3 - --skip-pull $ACR)

Kubernetes: Create imagePullSecret

ACR tokens are valid for 8 hours. For long-running workloads, you'll need to rotate.

ACR_NAME=dynamoci
ACR_REGISTRY=dynamoci.azurecr.io
ACR_TOKEN=$(az acr login --name $ACR_NAME --expose-token --output tsv --query accessToken)
kubectl create secret docker-registry acr-secret \
  --docker-server=$ACR_REGISTRY \
  --docker-username=00000000-0000-0000-0000-000000000000 \
  --docker-password=$ACR_TOKEN \
  --namespace <your-namespace>

Tag format reference

ComponentValues
commit_sha40-character git commit SHA
frameworkvllm, sglang, trtllm, dynamo, none
typedev, runtime, test, efa
cudacuda12, cuda13 (maps to 12.9, 13.x)

See available images per commit on the commit history dashboard (expand ImageSHA256 dropdown).
